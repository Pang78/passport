Passport Analyzer API

This is the API-only deployment of the Passport Analyzer application.

Available Endpoints

Health Check

GET /api/health

Check if the API is running correctly.

curl https://passport-analyzer.vercel.app/api/health

Extract Passport Data

POST /api/extract-passport

Extract data from a passport image using AI.

curl -X POST https://passport-analyzer.vercel.app/api/extract-passport \
  -F "image=@passport.jpg"

Note

This is an API-only deployment. For the full application, please visit the main website or run the application locally.
